Features

This is accent a minimal, mobile friendly theme for Jekyll

It features:

  1. Customizable accent
  2. Comments with Disqus
  3. Permalinks for headers
  4. Guest Posts
  5. Author’s Page
  6. Author’s Profile Picture, Short Bio.
  7. Pagination using jekyll-paginate
  8. Syntax Highlighting for code
  9. RSS Feed
  10. Google Analytics
  11. A 404 page.
  12. Full Documentation on the Demo website.
